# Searching, Querying, and Editing Stars

Use the search and data panels to find stars, filter results, and edit details.

## Search

1. Open the search panel.
2. Enter a star name, catalog ID, or filter criteria.
3. Apply the filter to update the plotted results.

## Editing Star Data

1. Select a star.
2. Open the details or edit panel.
3. Update fields and save changes.

## Notes

- Some fields may be read-only based on the dataset source.
- If edits do not persist, confirm the dataset is writable and backed by the local database.
