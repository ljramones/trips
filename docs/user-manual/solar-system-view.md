# Solar System View

The Solar System view lets you explore the planets around a selected star.

## Enter a Solar System

1. Select a star in the Interstellar view.
2. Use the context action or toolbar to enter the Solar System view.

## Controls

- Pan/zoom to examine orbits.
- Use the side pane to view planet lists and details.
- Adjust time speed to animate orbital motion.

## Notes

- Some stars may not have planetary systems defined.
- Procedural generation can be used to create missing systems.
