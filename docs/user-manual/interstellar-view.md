# Interstellar View

The Interstellar view is the primary 3D star field used for exploring, selecting, and plotting objects.

## Navigation

- Drag to rotate the view.
- Scroll to zoom in and out.
- Use the side pane to toggle overlays and display options.

## Selecting Objects

- Click a star to view its details.
- Use the object list to jump to specific entries.
- Right-click for context actions (if enabled).

## Display Options

Common options include label visibility, spectral class coloring, and grid overlays. These are available in the right-side pane.
