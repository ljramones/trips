# Route Planning

TRIPS includes route planning for interstellar travel paths between star systems.

## Plan a Route

1. Select a start star.
2. Select a destination star.
3. Open the route planner pane.
4. Configure distance limits and filters.
5. Generate routes and view results in 3D.

## Interpreting Results

- Routes are shown as lines with distance labels.
- Multiple alternatives may appear depending on your settings.

## Tips

- Tighten distance filters for shorter hops.
- Use dataset subsets for faster pathfinding on large catalogs.
