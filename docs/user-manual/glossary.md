# Glossary

- **Dataset**: A catalog of stars loaded into TRIPS for visualization and routing.
- **LOD (Level of Detail)**: Rendering technique that adjusts detail based on distance.
- **Transit**: A reachable jump between star systems based on distance or filters.
- **H2 Database**: Embedded local database used to store edits and preferences.
- **ACCRETE**: Procedural model for generating planetary systems.
